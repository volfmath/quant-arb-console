### 1.2 组件拆分

| 组件                        | Props / 数据来源                         | 说明                          |

| `AssetSummaryCard`         | `GET /api/dashboard/asset-summary`       | 资产总览 4 列 KPI             |

### 1.3 资产总览 KPI 定义

| 卡片         | 主数值                   | 副值                     | 数据字段                               |
|-------------|-------------------------|--------------------------|-----------------------------------------|
| 总资产估值   | USDT 总额 + `👁` 图标    | `≈ ¥` CNY 换算           | `total_equity`, `total_equity_cny`     |
| 今日收益     | USDT 绝对值（绿/红）     | 百分比变化                | `today_pnl`, `today_pnl_pct`           |
| 累计收益     | USDT 绝对值（绿/红）     | 百分比变化                | `total_pnl`, `total_pnl_pct`           |
| 可用资金     | USDT 绝对值              | 占总资产百分比            | `available_balance`, `available_pct`    |

### 1.5 策略表格列

| 列名         | 字段                 | 说明                                   |
|-------------|---------------------|----------------------------------------|
| 策略名称     | `name`              | 如"资金费套利_BTC"                     |
| 状态         | `status`            | Badge：运行中(绿) / 已暂停(灰)         |
| 运行时长     | `running_duration`  | 如"12天 07:32:16"                     |
| 今日收益     | `today_pnl`         | USDT，带颜色                           |
| 累计收益     | `total_pnl`         | USDT，带颜色                           |

| 胜率         | `win_rate`          | 百分比                                 |
| 最大回撤     | `max_drawdown`      | 百分比                                 |
| 操作         | —                   | "详情" 链接 + Toggle 开关              |

### 2.5 详情面板内容（分栏，非抽屉覆盖）

从上到下的内容区块：
1. **标题**：`BTC 资金费率套利详情` + 关闭按钮 `×`
2. **套利方向**：做多(Long) 交易所 ↔ 做空(Short) 交易所，卡片式展示
3. **KPI 行**：套利收益(8h) / 当前资金费率差 / 结算时间 / 下次结算
4. **费率趋势图**：24h 折线图，三条线（做多方费率、做空方费率、费率差）
5. **各交易所费率排名**：迷你表格（交易所 Logo + 费率 + 排名）
6. **底部按钮**：`⊕ 添加套利` 全宽主色按钮

### 3.3 任务卡片 5 区块详解

| 区块       | 内容                                                             | 样式                         |
|-----------|------------------------------------------------------------------|------------------------------|
| 标识区     | 图标(▶/✓) + `#编号` + 状态Badge + 交易所对 + 合约标识 + 时间      | 左对齐，合约名用 Caption 灰色  |
| 费率区     | 资金费率(含杠杆倍数) + 年化收益率                                 | 正绿负红                      |
| 仓位区     | 持仓数量(USDT) + 保证金(USDT)                                    | 白色数字                      |
| 交易区     | 买入(交易所 + 张数) + 卖出(交易所 + 张数)                        | 白色数字                      |
| 操作收益区 | 运行中=金额按钮(绿边)，已完成=closed(灰) + 收益/收益率           | 按钮 + 绿色收益               |

| 运行中   | ▶ (绿色) | 运行中   | 更新时间    | `[100 USDT]` 绿色边框   | 预估日收益 + 累计收益  |
| 已完成   | ✓ (灰色) | 已完成   | 结束时间    | `[closed]` 灰色         | 实际收益 + 收益率      |
| 异常     | ⚠ (红色) | 异常     | 异常时间    | `[处理]` 红色           | 最后收益 + 异常原因    |

### 3.6 任务状态机

```text
pending → confirming → running → closing → completed
                ↓          ↓                    ↑
             canceled    paused → running ──────┘
                           ↓
                        canceled
            running → failed (异常)
```

## 7. 通用交互规范

### 7.1 操作反馈

| 操作类型        | 反馈方式                                |
|----------------|----------------------------------------|
| 创建/编辑成功   | 顶部 Success Message，2s 自动关闭       |
| 操作失败        | 顶部 Error Message，需手动关闭           |
| 删除/停止确认   | Modal 弹窗二次确认                       |
| 表格/卡片加载   | 骨架屏（Skeleton）                       |
| 按钮提交中      | 按钮 Loading 态 + 禁用                  |
| WebSocket 断连  | 侧边栏 API 状态变红                     |

### 7.2 空状态

- 无数据时显示居中空状态插图 + 描述文字
- 首次使用引导：提示先配置交易所 → 再查看机会

### 7.3 错误处理

- API 401 → 跳转登录页
- API 403 → 显示无权限提示
- API 500 → 显示错误 Message + 重试按钮
- 网络断开 → 侧边栏 API 状态指示 + 页面内 Banner 提示
