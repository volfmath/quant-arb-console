## 关键路径

```text
B-09(交易所接入) → B-12(费率服务) → B-14(机会引擎) → B-19(执行引擎) → B-24(收益服务) → T-01(联调)
```

交易所接入层是最前置的依赖，应最先完成并充分测试。
